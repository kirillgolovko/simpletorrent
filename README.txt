Пояснение по сборке:
    Проект собирается с помощью цели package на проекте simpletorrent с помощью Maven.
    После этого в папках simpletorrent/client/target и simpletorrent/server/target
    окажутся jar-файлы для запуска со всеми зависимостями (server-1.0.0-jar-with-dependencies.jar
    и client-1.0.0-jar-with-dependencies.jar). Они готовы к самостоятельной работе вне проекта

Пояснение по запуску:
    Клиентская программа запускается двойным кликом или из командной строки
    Серверу нужно на вход передать путь xml-файла параметров, такой имеется в архиве - params.xml
    Список параметров с поясненениями:
    <appParams>
      <ServerPort>8080</ServerPort>
      <SharedDirPath>/home/kirill/Projects/simpletorrent/server/testDirectory</SharedDirPath>
      <maxClients>16</maxClients>
      <clientKeepAlive>3</clientKeepAlive>
      <blocksPerTransmission>64</blocksPerTransmission>
    </appParams>

    *ServerPort - порт, на котором сервер принимает соединения
    *SharedDirPath - путь к общей папке
    *maxClients - максимальное число одновременно подключенных клиентов
    *clientKeepAlive - сколько ждем ответа от клиента до закрытия сокета в секундах
    *blocksPerTransmission - сколько блоков по 512 байт передаем за 1 раз

Пояснения по использованию:
    При подключении:
        Указать адрес и порт сервера, clientId хоть и предусмотрен архитектурой, пока никак не используется
        (только выводится в лог на сервере)
        Нажать кнопку Connect
    При работе:
        Можно перемещаться по локальным каталогам справа и по каталогам на сервере в пределах общей папки слева.
        При нажатии на папку она открывается, а при нажати кнопки up происходит переход на каталог вверх
        Для копирования файла необходимо выделить желаемый файл (кликнуть по нему) и нажать кнопку =>
        Снизу выводится прогресс копирования.
        Для отключения - нажать кнопку Disconnect
